介绍 FCL
keyboard_arrow_down
这是什么？
keyboard_arrow_down

《Fold Craft Launcher》（下文简称FCL），此软件可以使你在Android设备上运行《我的世界：Java版》（《Minecraft : Java Edition》），并支持各类模组加载器。除此之外，还有自动安装、管理多个游戏版本和下载模组、整合包等多个强大功能。

这里是FCL玩家自己搭建的FCL发行版下载站。由于GFW阻断了FCL的GitHub链接，导致了许多玩家无法下载FCL最新版。我们提供了快速安全的直链。

游戏版本管理
keyboard_arrow_down

具有自动下载并安装游戏版本，支持模组加载器，并带有游戏版本管理。

内置资源下载
keyboard_arrow_down

除了下载游戏版本，还有下载整合包、模组、资源包、世界和光影。

兼容性强
keyboard_arrow_down

目前已内置支持Java 8、Java 11、Java 17和Java 21，具有Pojav与Boat两种启动方式。启动器会根据游戏版本自动选择Java版本。

自制或者在布局仓库下载其他玩家制作的控制布局，没有实体键盘也可以获得不错的游戏体验。
